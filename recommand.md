# 建議事項

1. 目錄下的主要檔案或第一個要執行的檔案儘量以index.php或index.html來命名
2. 儘量使用相對路徑來取代絕對路徑
```
   <a href="per.php?m=......
    改成
   <a href="?m=.....
```
3. 網址列後的查詢字串中儘量不要有空白，避免不同網頁伺服器在解析時發生錯誤
```
<a href="?m=<?=($month-1);?> & y=<?=($year);?>">上一月</a>
去掉&前後的空白改成
<a href="?m=<?=($month-1);?>&y=<?=($year);?>">上一月</a>
```
4. 建議背景的圖片刷淡或降低色調，避免紅色的的部份和月曆的紅色字重疊而看不清
